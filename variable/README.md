# def var val

def는 method를 선언하기 위해서 사용, 써보니까 변수도 설정되네

var는 type변경 가능

val는 type변경 불가// 초기화 후에 재할당 불가

변수 선언할때 뒤에 ; 써도 되고 안써도되고

값을 할당을 안하고 그냥 단순한 정의는 안되는듯...?
